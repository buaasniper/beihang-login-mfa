# beihang login MFA

Last updated: 2026-04-13

## 项目定位

本目录用于整理“北航登录 MFA 智能风险平台”的系统蓝图、研究主线与阶段性推进材料，作为后续工程搭建、研究论证、周报跟踪和对外展示的统一入口。

当前版本聚焦于以下目标：

- 明确 Online / Offline / Research 的统一系统结构
- 明确 Anti-Bot 先于 Device-ID / RBA 的执行逻辑
- 明确 `device-id v1 -> v2` 的升级路线
- 明确 Offline intelligence 动态更新 Online 的闭环
- 明确 research validation、shadow evaluation 和 rule lifecycle 的后续扩展方向

- `研究内容与创新蓝图.md`：产品特点、核心模块解释、输入输出、原因与创新点
